## Hashtags Battle
> PLAYGROUND. Real-time Twitter's hashtags analytics. 

### About
Consumes thousands of inputs per seconds and does daily/hourly aggregation on hashtags.

Inspired by [One Million Tweet Map](https://onemilliontweetmap.com/)
